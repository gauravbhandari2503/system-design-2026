# System Design Interview Questions

This section contains a collection of common frontend system design interview questions. Each question is accompanied by a reference implementation in the codebase.

## How to Use This Resource

For each question below, we have provided a complete reference solution in the project repository. Inside each folder (e.g., `Question 1`), you will find:

### 1. Documentation (`/docs`)

- **Basic Idea** (`docs/basic-idea.md`): Read this **first**. It explains the high-level architectural approach, data structures, component hierarchy, and key trade-offs. It answers the "Why".
- **Implementation Plan** (`docs/implementation-plan.md`): This outlines the step-by-step plan to build the solution, useful for understanding how to break down the problem during an interview.

### 2. Working Code (`/src`)

A fully functional Vue.js implementation of the problem.

### Running the Examples Locally

To run any of these examples on your machine:

1.  Open your terminal.
2.  Navigate to the specific question folder:
    ```bash
    cd "Question 1"
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```
5.  Open the localhost URL provided (usually `http://localhost:5173`).
